# SELinuxOFF
This is a script that allows you to turn off SELinux in 10 seconds

## Usage
      wget https://raw.githubusercontent.com/CY031102/SELinuxOFF/main/SELinuxOFF.sh
      chmod +x SELinuxOFF.sh
      ./SELinuxOFF.sh
