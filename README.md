# SELinuxOFF
A script that allows you to turn off SELinux in 10 seconds

Made for CentOS 7

## Usage
      wget https://raw.githubusercontent.com/CY031102/SELinuxOFF/main/SELinuxOFF.sh
      chmod +x SELinuxOFF.sh
      ./SELinuxOFF.sh
